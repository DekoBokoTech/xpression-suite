このツールの使い方・育て方How to use & grow this tool

基本の考え方：Visual Logicは「オブジェクトブロック（シーン/オブジェクト/マテリアルのパラメータ）」と「ファンクションブロック（Logic・Math・Timer など）」をワークスペースに置き、黄色い出力点 → 緑の入力点へ線をつないで動作を組み立てます。このツールは「やりたいこと」からその組み合わせを逆引きするための索引です。

ブロック種別の色：

Colors(赤) Data Sources(淡緑) Date Time(紫) Generators(青) Logic(青紫) Math(緑) Selectors(橙) Strings(マゼンタ) Timers/Counters(深緑) Vector Math(黄) Object(灰) Widget(ウィジェット) 外部制御(RossTalk/GPI/PBus)

配線ルール：緑の線＝つなげる／赤い矢印＝つなげない／青い矢印＝データ共有中。左の緑ドット＝入力、右の黄ドット＝出力。

育て方：「＋ レシピを追加」で自分のノウハウを足せます（この端末に保存）。共有したい時は「配布用HTML」で追加分を本体に埋め込んだファイルを書き出せます。

The idea: In Visual Logic you place object blocks (scene/object/material parameters) and function blocks (Logic, Math, Timer, …) on a workspace and connect yellow output → green input to build behavior. This tool is a reverse index from what you want to do to that combination.

Block category colors:

Colors (red) Data Sources (mint) Date Time (purple) Generators (blue) Logic (blue-purple) Math (green) Selectors (orange) Strings (magenta) Timers/Counters (dark green) Vector Math (yellow) Object (gray) Widget External Ctrl (RossTalk/GPI/PBus)

Wiring rules: green line = connectable / red arrow = not / blue arrow = sharing data. Left green dot = input, right yellow dot = output.

Growing it: use "+ Add recipe" to add your own know-how (saved on this device). To share, use "Export HTML" to bundle your recipes into a distributable file.
